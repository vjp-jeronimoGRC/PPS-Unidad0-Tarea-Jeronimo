## Configuración del GitHub Pages

Para publicar la documentación, voy a Settings > Pages y selecciono:

```
Source: GitHub Actions
```

## Creacion automática de la rama gh-pages

La nueva rama será:

```
gh-pages
```

## Trabajar la rama gh-pages
- El workflow de GitHub Actions genera automáticamente la rama `gh-pages`.
- Esta rama contiene todos los archivos de la documentación
